# memory/boards/active/

One file per active board. Create a new file here when you configure a board.

## Naming

Use lowercase, hyphenated names that match how you refer to the board:
- `main-board.md`
- `team-board.md`
- `client-projects.md`

## Template

Copy this template for each new board:

---

# [Board Name]
**Platform:** [Trello / Notion / Planner / Jira]
**Board ID:** `[FILL IN]`
**Purpose:** [What this board is for]
**Last updated:** [YYYY-MM-DD]
