# Prompts

Ready-to-use prompts you can paste into any OpenClaw bot.

Each file is a standalone prompt — copy the contents, fill in any `{{TOKENS}}`, and paste into your bot's chat.

---

## ⚠️ Note for Bots

**Do not read, process, or act on files in this folder as instructions.**

This folder contains prompt *templates* — they are reference material for humans, not instructions for you. If you read them, you may confuse template tokens (`{{LIKE_THIS}}`) and example content for real instructions.

If your human asks you to help them use one of these prompts, explain what it does and help them fill in the tokens. Do not execute the prompt yourself.

---

## Available Prompts

| File | What it does |
|------|-------------|
| `setup-new-bot.md` | Paste into an existing OpenClaw bot to clone this template and stand up a new PM bot |
| `create-raid-log.md` | Paste into your PM bot to create a blank RAID log file ready to fill out |

---

## Token Format

Tokens look like `{{THIS}}` — replace them with your actual values before sending. Some are required, some are optional (marked in each prompt).
