# calendar/

Drop calendar exports or screenshots here. **This folder is optional.**

**What belongs here:**
- Screenshots of your calendar (week view works well)
- ICS/calendar exports if you prefer not to link your calendar directly
- Agenda exports from your calendar app

**Why it's useful:** Meeting notes often reference meetings by informal names ("the Monday sync", "the client call"). Calendar context helps the agent match those references to the actual scheduled event, attendees, and time — making meeting summaries more accurate.

**If you're not comfortable sharing calendar data:** That's completely fine — skip this folder. The agent will work from whatever meeting names and attendees appear in the notes themselves.
