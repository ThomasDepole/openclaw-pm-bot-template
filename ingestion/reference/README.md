# reference/

Drop general context documents here — anything that doesn't fit neatly in another folder.

**What belongs here:**
- Company overview and about docs
- Mission, vision, and values documentation
- Industry or market context docs
- Glossaries and terminology guides
- Competitive landscape notes
- Historical background or company history
- Anything you want the agent to know that doesn't belong elsewhere

**Tip:** If you're not sure where something goes, put it here. The agent will extract what's useful and file it in the right memory category.
