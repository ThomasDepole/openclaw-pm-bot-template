# status-reports/

Drop project and sprint status updates here.

**What belongs here:**
- Weekly or bi-weekly status reports
- Sprint reviews and retrospectives
- Project health dashboards (exports or screenshots)
- Burn-down or burn-up charts with context notes
- Milestone updates sent to stakeholders

**Tip:** Even a simple "here's where things stand" email or Slack message counts. If it tells the story of where a project is right now, drop it here.
