# people/

Drop team and stakeholder information here.

**What belongs here:**
- Team rosters and org charts
- Stakeholder lists (internal and external)
- Role descriptions and responsibilities
- Contractor or vendor contacts
- Bios or background notes on key people

**Tip:** Even a simple spreadsheet with Name / Role / Email / Project is enough to get started. The agent will build on it from there as it sees names appear in meetings and status reports.
