# meetings/

Drop meeting notes here.

**What belongs here:**
- Meeting summaries and call notes
- Daily stand-up / scrum notes
- Client call recaps
- AI-generated transcripts (Plaud, Otter.ai, Fireflies, etc.)
- Any written record of a conversation that had decisions or action items

**Tip:** Files don't need to be clean. Raw AI transcripts, rough bullets, even a voice memo transcript — drop it and let the agent distill it.
